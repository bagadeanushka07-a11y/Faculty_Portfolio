## ✨ Features

### 🎨 Design Features

- ✅ **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- ✅ **Dark/Light Mode** - User preference saved in localStorage
- ✅ **Glassmorphism Effects** - Modern, premium glass UI elements
- ✅ **Smooth Animations** - Scroll reveal animations and transitions
- ✅ **Sticky Navigation** - Fixed navbar with active section highlighting
- ✅ **Preloader Animation** - Professional loading screen

### 💻 Functional Features

- ✅ **Profile Photo Upload** - Upload and save profile picture (localStorage)
- ✅ **Animated Statistics Counters** - Numbers animate when scrolled into view
- ✅ **Contact Form** - Integrated with EmailJS for email sending
- ✅ **Auto-Reply Email** - Automatic confirmation email to users
- ✅ **Subject-wise Materials** - Organized resources for each subject
- ✅ **Downloadable Resources** - Lecture notes, PPTs, Question Banks, Lab Manuals
